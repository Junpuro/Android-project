# Book Management App

## Description

A simple Android application for managing books.

## Features

* Add, edit, delete books
* Categorize books by genre

## Tech Stack

* Java / Kotlin
* Android Studio
* SQLite

## How to run

* Clone the project
* Open in Android Studio
* Run on emulator or device
